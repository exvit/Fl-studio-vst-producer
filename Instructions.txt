Plugins supplémentaires de FL Studio 20.9.1.
================================================
Le dossier "Addition Plugins from version 20.9.1" contient le programme d'installation des plugins supplémentaires Pitch Shifter et Vintage Chorus de la version 20.9.1.
Les plugins ne sont installés que pour la version 20.8.4.2576 et ne fonctionneront qu'avec les fichiers FLEngine corrigés dans la version 64 bits de FL Studio.
Si plusieurs versions de FL Studio sont installées sur le système, FL Studio 20.8.4.2576 Plugins Setup.exe doit être exécuté à partir du dossier avec la version 20.8.4.2576

Attention! Dans la version officielle de FL Studio 20.8.4, ces plugins ne sont pas disponibles, en les installant vous êtes conscient de tous les risques.
